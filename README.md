# Encriptador de Texto

Una sencilla página web que permite **encriptar** y **desencriptar** mensajes ingresados por el usuario, usando reglas de sustitución específicas. El objetivo es brindar una forma divertida de ocultar y recuperar mensajes con un estilo único.

## 🌐 Ver la Página Web

Puedes acceder a la página del aplicativo en el siguiente enlace:  
[Encriptador de Texto - Challenge ONE](https://user-name-c.github.io/challengeEncriptadorONE/)

## 🛠️ Tecnologías utilizadas

- **HTML**  
- **CSS**  
- **JavaScript**

## 🚀 Funcionalidades

- **Encriptar**: Transforma el texto ingresado utilizando las siguientes reglas:
  - La letra **"e"** se convierte en **"enter"**.
  - La letra **"i"** se convierte en **"imes"**.
  - La letra **"a"** se convierte en **"ai"**.
  - La letra **"o"** se convierte en **"ober"**.
  - La letra **"u"** se convierte en **"ufat"**.

- **Desencriptar**: Revertir el mensaje encriptado para recuperar el texto original.

## 📋 Instrucciones de Uso

1. **Abrir la página web** en el navegador.
2. Escribir el **mensaje** que deseas encriptar o desencriptar en el área de texto.
3. Hacer clic en el botón **"Encriptar"** o **"Desencriptar"** según lo que necesites.
4. El resultado se mostrará automáticamente en la pantalla.

## 🎨 Diseño

El proyecto utiliza **HTML** para la estructura, **CSS** para el estilo visual, y **JavaScript** para la lógica de encriptación y desencriptación.
